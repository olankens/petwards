  <h1>PETWARDS</h1>
  <p>Petwards turns shelter management into a magical experience for fantastic beasts. Its JWT-secured Spring Boot API manages creature records, an adoption system, and advanced filtering through a clean setup.</p>

<hr>

### Main Features

- Anonymous wizards can:
  - List available beasts
  - Filter beasts by their capabilities
  - Register as adopter
- Adopter wizards can:
  - Ask for beast adoption
- Staff wizards can:
  - Approve or reject adoptions
  - Create beasts with capabilities
- Admin wizards can:
  - Create beasts with capabilities
  - Create staff wizards
  - Update shelter information
